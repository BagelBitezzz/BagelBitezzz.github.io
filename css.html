CSS Basics

What's CSS?

CSS is a language used for describing the presentation of a document written with a markup language such a HTML.

CSS stands for Cascading Style Sheets, and is the laguage we use to style Web Pages. CSS describes how HTML elements are displayed on a website, allowing you to have unique websites easily.

What Is The History Of CSS?

In October 10, 1994, Hakon Wium Lie proposed a new style sheet language called CSS. A main feature of CSS was that it let a document's style be influenced by multiple style sheets by way of "cascading" styles.
