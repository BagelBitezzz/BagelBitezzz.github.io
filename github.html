GitHub is a hosting platform for version control and collaboration on coding projects.

It lets you and others work together on projects from anywhere and at any time. These basics teach you GitHub essentials. Click the button below for Github's main page.

Github's History

On October 19, 2007, development on the github platform started. In April 2008 the site had its official launch by developers Tom Preston-Werner, Chris Wanstrath, P. J. Hyett and Scott Chacon, after a short beta period of testing.
