HTML Basics

What Is HTML?

HTML is the standard markup language for documents designed to be displayed in a web browser.

HTML stands for Hyper Text Markup Language, and it is the standard language for building webpages. HTML consists of elements that allow a browser to properly display content on a page. When Was HTML Created?

The first ever version of HTML was written in 1993 by Tim Berner-Lee. Since then, HTML has had many versions, but in 1999 HTML version 4.01 became the official standard, and throughout the 2000's this version was the most widely used. Now HTML5 is the latest major version!
