What Does CSS Do Exactly?

Once you have coded a page with HTML it still is missing something. CSS is missing. CSS is what makes pages look amazing. CSS can control the size of things, where things are, and how much space between other things. Heres an example!

body {
  background-color: lightblue;
}

h1 {
  color: white;
  text-align: center;
}

p {
  font-family: verdana;
  font-size: 20px;
}
